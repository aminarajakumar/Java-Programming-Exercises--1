# Java Fundamentals and Object-Oriented Programming

## Overview

This repository contains Java programs developed to practice and understand core Java concepts. The programs cover fundamental programming constructs, object-oriented programming principles, type conversion, constructors, encapsulation, and inheritance.

The project is organized into multiple parts, with each part focusing on a specific Java concept.

## Topics Covered

### Part A – Java Basics

* Printing output
* Understanding the main() method
* Multiple classes in a single file
* Primitive data types
* Wrapper class constants

### Part B – Type Conversion

* Widening conversion
* Narrowing conversion
* Data loss during type casting

### Part C – Operators and Calculations

* Marks and percentage calculation
* Employee salary calculation
* Number swapping techniques

### Part D – Classes and Objects

* Creating classes
* Object creation
* Methods and data members

### Part E – Constructors

* Constructor initialization
* Constructor overloading
* Using the this keyword

### Part F – Encapsulation

* Private variables
* Getter and Setter methods
* Bank Account implementation

### Part G – Inheritance

* Single inheritance
* Hierarchical inheritance
* Method inheritance
